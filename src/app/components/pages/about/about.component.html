6 ways to center inner element


[absolute] center inner by absolute + transform, translate


.outer => position: relative;
.inner => position: absolute; top: 50%; left: 50% transform: translate(-50%, -50%);

[absolute 2] center inner by absolute + margin auto + top, right same offset & bottom, left same offset


offset can be a range of numbers [0, x], usually set to 0


[flex] center one child by set justify-content, align-items of outer


only set outer element


[flex 2] center one child by flex + margin auto of inner


.outer => display: flex;
.inner => margin: auto;

[table-cell] center one child by table-cell + text-align +vertical-align


only set outer element

note: seems margin not working on 'table-cell'


[grid] outer display grid + inner align-self, inner justify-self


.outer => display: grid;
.inner => align-self: center; justify-self: center;
